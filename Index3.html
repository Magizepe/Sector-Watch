📊 How live prices work

  • Every time you open the site, prices are fetched automatically from all active sources
  • Sources are tried in order: Finnhub → Twelve Data → Alpha Vantage → Yahoo Finance → FMP
  • The first source that returns a valid price wins for that ticker
  • Prices also auto-refresh every 5 minutes while the site is open
  • If all live sources fail, calibrated war-period data is shown as a fallback
  • The Source column in the dashboard table shows Live or Calibrated per stock
